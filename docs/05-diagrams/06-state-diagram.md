│                                                                       │

│  (●) ────────────▶ ┌─────────────┐                                 │
│                     │  INACTIVE   │                                 │
│                     │ (unverified)│                                 │
│                     └──────┬──────┘                                 │
│                            │                                         │
│                 verifyOTP()│                                         │
│                            ▼                                         │
│                     ┌─────────────┐                                 │
│              ┌─────▶│   ACTIVE    │◀──────────────┐                 │
│              │      └──────┬──────┘               │                 │
│              │             │                      │                 │
│              │    ┌────────┼───────────┐          │                 │
│              │    │        │           │          │                 │
│              │    │ lock() │ delete()  │ adminBan()                  │
│              │    │(5 fails)│          │          │                 │
│              │    ▼        ▼           ▼          │                 │
│              │ ┌────────┐ ┌────────┐ ┌────────┐  │                 │
│   unlock()   │ │ LOCKED │ │DELETED │ │ BANNED │  │                 │
│  (after 30m) │ │(temp)  │ │(soft)  │ │        │  │                 │
│              │ └───┬────┘ └────────┘ └───┬────┘  │                 │
│              │     │          │          │        │                 │
│              │     │          ▼          │ unban()│                 │
│              └─────┘        (◉)         └────────┘                 │
│                                                                       │
│  Guard Conditions:                                                   │
│  • INACTIVE → ACTIVE: OTP verified (email OR phone)                 │
│  • ACTIVE → LOCKED: 5 consecutive failed login attempts             │
│  • LOCKED → ACTIVE: 30 minutes elapsed (auto) or admin unlock      │
│  • ACTIVE → DELETED: User requests deletion + 30 day grace period   │
│  • ACTIVE → BANNED: Admin action (fraud, abuse)                     │
│  • BANNED → ACTIVE: Admin unban                                     │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
